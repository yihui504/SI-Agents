## 6. 移植建议

### 高优先级
1. **实现优化循环**：移植 SkVM 的 loop.ts 逻辑，支持多轮优化迭代
2. **实现工作区管理**：移植 workspace.ts，支持临时工作区创建与清理
3. **集成 NanobotPolicy**：将新实现的执行命令策略集成到优化流程

### 中优先级
1. **移植 runtime 模块**：实现运行时调度功能
2. **完善 profiler**：移植性能分析能力

### 低优先级
1. **缓存管理**：实现技能执行结果缓存
2. **特定场景优化器**：针对特定语言和场景的优化

## 7. 当前状态总结

- **已移植功能**：优化器框架、安全扫描、约束注入、验证器
- **未移植功能**：优化循环、工作区管理、runtime、完整 profiler
- **新增功能**：完整的安全策略系统（来自 ArbiterOS）
- **测试覆盖**：基础功能有测试，优化循环等缺失功能无测试
